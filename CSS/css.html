¿Qué es CSS?

CSS (Hoja de Estilos en Cascada) es un lenguaje que describe cómo se deben presentar los elementos de un documento HTML. Permite controlar el diseño, colores, fuentes y espaciado de una página web.

Cómo aplicar CSS

Existen tres maneras principales de aplicar CSS:

  • CSS en línea: Se aplica directamente en la etiqueta HTML mediante el atributo style.
  • CSS interno: Se incluye dentro de la misma página HTML mediante la etiqueta <style>.
  • CSS externo: Se enlaza un archivo CSS separado mediante la etiqueta <link>.

Ejemplos

1. CSS en línea

Este es un párrafo con estilo en línea (color azul y negrita).

2. CSS interno

Usa la etiqueta <style> en el <head> de tu documento HTML para agregar estilos.


<style>
p {
color: green;
font-size: 18px;
}
</style>

3. CSS externo

Recomendado para proyectos grandes, ya que permite separar el contenido del diseño.


<link rel="stylesheet" href="estilos.css">

Ejemplos prácticos

Cambiar el color de fondo

Este ejemplo cambia el color de fondo de la página a un tono claro:


body {
    background-color: #f4f4f4;
}

Estilizar un botón

Aquí mostramos cómo darle estilo a un botón:


button {

    background-color: #008cba; /* Color de fondo del botón */

    color: white; /* Color del texto */

    border: none; /* Sin borde */

    padding: 10px 20px; /* Espaciado interno */

    font-size: 16px; /* Tamaño de la fuente */

    border-radius: 5px; /* Bordes redondeados */

    cursor: pointer; /* Cambia el cursor al pasar por encima */

}



button:hover {

    background-color: #005f7a; /* Color de fondo al pasar el mouse */

}

Estilizar una imagen


.styled-image {
    width: 300px; /* Ajusta el ancho */
    height: auto; /* Mantén la proporción */
    border: 10px solid rgb(196, 61, 61); /* Borde alrededor de la imagen */
    border-radius: 10px; /* Bordes redondeados */
    box-shadow: 10px 10px 10px rgb(66, 20, 20); /* Sombra */
}

Estilizar una barra de navegación


nav.styled-nav {
    display: flex; /* Activa el modelo flexible */
    justify-content: space-around; /* Espaciado equitativo entre elementos */
    align-items: center; /* Alinea verticalmente los elementos */
    background-color: #333;
    padding: 10px 0px;
}

nav.styled-nav a {  /* Estilizacion del link */
    color: white; /* Color del texto */
    text-decoration: none; /* Quita la decoracion del link */
    padding: 5px 15px;
}

nav.styled-nav a:hover {
    background-color: #555;
    border-radius: 5px;
}

Padding

Es el espacio interno entre el contenido de un elemento y su borde.(Representado en amarillo)
Se usa cuando quieres crear espacio dentro de la caja del elemento, es decir, entre el contenido (texto, imagen, etc.) y el borde del elemento.

Margin

Es el espacio externo entre el borde del elemento y los elementos adyacentes.(Representado en rojo)
Se usa cuando quieres separar un elemento de otros elementos cercanos.

Propiedades comunes de CSS

  • color: Define el color del texto.
  • background-color: Cambia el color de fondo de un elemento.
  • font-family: Establece la familia de fuentes del texto.
  • font-size: Define el tamaño del texto.
  • padding: Controla el relleno interno de un elemento.
  • margin: Controla el margen externo de un elemento.
  • border: Agrega un borde alrededor de un elemento.
  • text-align: Alinea el texto (izquierda, derecha, centro o justificado).
