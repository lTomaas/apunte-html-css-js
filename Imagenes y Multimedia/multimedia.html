Imágenes en HTML

Sintaxis Básica


            <img src="ruta_de_imagen" alt="descripción de la imagen" title="titulo de la imagen">
  • src: Especifica la fuente o ruta del archivo de imagen. Puede ser una ruta relativa o absoluta. Por ejemplo:
    • Ruta relativa:"imagenes/foto.jpg"
    • Ruta absoluta:"https://www.ejemplo.com/imagenes/foto.jpg"
  • alt: Proporciona una descripción alternativa de la imagen. Es importante para accesibilidad y SEO. Si la imagen no se carga, el texto alternativo será visible.
  • title: Proporciona un texto emergente cuando el usuario pasa el cursor sobre la imagen.

Multimedia en HTML

Audio

La etiqueta <audio> permite incluir archivos de audio en una página web. Se puede usar con el atributo controls para proporcionar controles básicos de reproducción. 


            <audio controls>
                <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mp3">
                Tu navegador no soporta la etiqueta de audio.
            </audio>
  • type: Especifica el tipo MIME del archivo, como audio/mp3, audio/ogg, etc.
  • controls: Muestra los controles básicos de reproducción como play, pause, volumen, etc.
  • autoplay: Hace que el audio se reproduzca automáticamente cuando la página se carga.
  • loop: Hace que el audio se repita infinitamente.
  • muted: Mutea el audio al inicio.

Video

La etiqueta <video> permite incluir archivos de video en una página web. Al igual que con el audio, puedes usar el atributo controls para mostrar los controles de reproducción. 


            <video controls width="600">
                <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
                Tu navegador no soporta la etiqueta de video.
            </video>
  • type: Especifica el tipo MIME del archivo, como video/mp3, video/ogg, etc.
  • controls: Muestra los controles básicos de reproducción como play, pause, volumen, etc.
  • autoplay: Reproduce el video automáticamente cuando la página se carga.
  • loop: Hace que el video se repita infinitamente.
  • muted: Mutea el video.
  • poster: Muestra una imagen previa al video mientras se carga o antes de iniciar la reproducción.
