Estructura Basica de un Formulario 


            <form action="/ruta_servidor" method="POST"> 
                <label for="nombre">Nombre:</label>
                <input type="text" id="nombre" name="nombre" required>
                <input type="submit" value="Enviar">
            </form>
  • action: Define la URL a la que se envía el formulario.
  • method: Especifica el método HTTP para enviar los datos (GET o POST).
  • name: Asigna un nombre al campo para identificarlo en el servidor.
  • required: Indica que el campo es obligatorio.

Tipos de entradas (input) más comunes

Texto (text) 


            <input type="text" name="nombre" placeholder="Ingresa tu nombre">

Contraseña (password)


            <input type="password" name="clave" placeholder="Ingresa tu contraseña">

Correo electrónico (email)


            <input type="email" name="correo" placeholder="Ingresa tu correo" required>

Número (number)


            <input type="number" name="edad" min="1" max="100">

Fecha (date)


            <input type="date" name="fecha_nacimiento">

Selector de archivos (file)


            <input type="file" name="archivo">

Casilla de verificación (checkbox)

Acepto los términos 

            <input type="checkbox" name="acepto" value="si" required> Acepto los términos

Botón de radio (radio)

Masculino Femenino 

            <input type="radio" name="genero" value="masculino"> Masculino
            <input type="radio" name="genero" value="femenino"> Femenino

Botón de envío (submit)


            <input type="submit" value="Enviar">

Elementos avanzados

Área de texto (textarea)


            <label for="mensaje"> Mensaje: </label>
            <textarea id="mensaje" name="mensaje" rows="4" cols="50"> </textarea>

Selector desplegable (select)


    <label for="pais"> País: </label>
    <select id="pais" name="pais">
        <option value="argentina"> Argentina </option>
        <option value="mexico"> México </option>
        <option value="espana"> España </option>
    </select>

Botón (button)


            <button type="submit"> Enviar </button>

Atributos comunes de formularios

  1. placeholder: Muestra un texto de ejemplo dentro del campo.
  2. value: Especifica el valor inicial del campo.
  3. readonly: Hace el campo de solo lectura.
  4. disabled: Desactiva el campo.
  5. autocomplete: Controla si el navegador sugiere autocompletado.
  6. maxlength: Establece el número máximo de caracteres permitidos.
