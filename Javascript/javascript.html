Variables y tipos de datos

Variables

  1. var

    Declaración global o de función (no recomendada debido a problemas de alcance).

  2. let

    Declaración de bloque (recomendada para variables que pueden cambiar).

  3. const

    Declaración de bloque para valores constantes (recomendada para valores que no cambian).


let nombre = "Juan"; 
const PI = 3.14; 
var edad = 25;

Tipos de Datos Primitivos

  1. String

    Cadena de texto.
    Ej: "Hola", "Mundo"

  2. Number

    Números (enteros y decimales).
    Ej: 42, 3.14

  3. Boolean

    Verdadero o falso.
    Ej: true, false

  4. Undefined

    Variable declarada pero sin valor asignado.

  5. Null

    Valor intencionalmente vacío.

  6. Symbol

    Identificador único e inmutable.

  7. BigInt

    Enteros mayores que 2^53 - 1.

Estructuras de Control

Condicionales


if (condicion) {
    // Código si la condición es verdadera
} else if (otraCondicion) {
    // Código si la otra condición es verdadera
} else {
    // Código si ninguna condición se cumple
}

Switch


switch (expresion) {
case valor1:
    // Código
    break;
case valor2:
    // Código
    break;
default:
    // Código por defecto
}

Bucles

  1. for

    Repite un bloque de código un número determinado de veces. 

    2. 
for (let i = 0; i < 10; i++) {
    console.log(i);
}

  2. while

    Repite un bloque de código mientras se cumpla una condición.

    3. 
let i = 0;
while (i < 10) {
    console.log(i);
    i++;
}

  3. do...while

    Similar a while, pero siempre ejecuta el bloque al menos una vez. 

    4. 
let i = 0;
do {
    console.log(i);
    i++;
} while (i < 10);

  4. for...of

    Itera sobre elementos de un iterable (arrays, strings, etc.).

    5. 
for (let valor of array) {
    console.log(valor);
}

  5. for...in

    Itera sobre las propiedades de un objeto.

    6. 
for (let propiedad in objeto) {
    console.log(propiedad);
}

Funciones

Declaración de Funciones

Este es el modo clásico de definir una función. Se puede invocar antes o después de ser definida gracias al hoisting (las funciones declaradas se elevan al inicio del ámbito). 


function saludar() {
    console.log("Hola");
}
saludar(); // Llama a la función y muestra "Hola"
  1. ¿Qué hace?

    Imprime un saludo en la consola cada vez que se invoca.

Funciones anónimas

Son funciones que no tienen nombre y se suelen asignar a una variable. No pueden ser invocadas antes de su definición. 


const sumar = function (a, b) {
    return a + b;
};
console.log(sumar(5, 3)); // Devuelve 8
  1. ¿Qué hace?

    Recibe dos números y devuelve su suma.

Arrow functions (funciones de flecha)

Introducidas en ES6, son una forma más compacta de escribir funciones. No tienen su propio this, lo que las hace ideales para callbacks. 


const multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 5)); // Devuelve 20
  1. ¿Qué hace?

    Recibe dos números y devuelve su producto.

Funciones con parámetros predeterminados

Permiten definir valores predeterminados para los parámetros, que se usan si no se proporciona un valor al invocar la función. 


function saludar(nombre = "Amigo") {
    console.log(`Hola, ${nombre}`);
}
saludar();           // Muestra "Hola, Amigo"
saludar("Carlos");   // Muestra "Hola, Carlos"
  1. ¿Qué hace?

    Imprime un saludo personalizado. Si no se pasa un nombre, usa "Amigo" como valor predeterminado.

Funciones de orden superior

Son funciones que pueden recibir otras funciones como parámetros o devolverlas como resultado. Son clave en la programación funcional. 


function procesarArray(arr, callback) {
    const resultado = [];
    for (let i = 0; i < arr.length; i++) {
        resultado.push(callback(arr[i]));
    }
    return resultado;
}
const duplicar = (n) => n * 2;
console.log(procesarArray([1, 2, 3], duplicar)); // Devuelve [2, 4, 6]
  1. ¿Qué hace?

    Recibe un array y una función (callback), aplica la función a cada elemento del array y devuelve un nuevo array con los resultados.

Objetos

  1. Objetos literales

let coche = {
    marca: "Toyota",

  1. Clases y herencia

class Persona {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad = edad;
    }
    
    saludar() {
        console.log(`Hola, soy ${this.nombre}`);
    }
}
    
class Estudiante extends Persona {
    constructor(nombre, edad, carrera) {
        super(nombre, edad);
        this.carrera = carrera;
    }
}

Manipulacion del DOM

Métodos principales de Seleccion

  1. getElementById

    Selecciona un elemento por su id.

    2. 
const titulo = document.getElementById("miTitulo");
console.log(titulo.innerText);
  2. getElementsByClassName

    Devuelve una colección de elementos con una clase específica.

    3. 
const items = document.getElementsByClassName("item");
console.log(items[0]); // Primer elemento con clase "item"
  3. getElementsByTagName

    Devuelve una colección de elementos con una etiqueta específica.

    4. 
const divs = document.getElementsByTagName("div");
console.log(divs.length); // Número de divs en el documento
  4. querySelector

    Selecciona el primer elemento que coincida con un selector CSS.

    5. 
const primerParrafo = document.querySelector("p");
  5. querySelectorAll

    Devuelve todos los elementos que coinciden con un selector CSS.

    6. 
const enlaces = document.querySelectorAll("a");
enlaces.forEach((enlace) => console.log(enlace.href));

Modificar el contenido de los elementos

  1. innerText

    Cambia el texto que se muestra dentro de un elemento.

    2. 
const titulo = document.getElementById("miTitulo");
titulo.innerText = "Nuevo Título";
  2. innerHTML

    Cambia el contenido HTML dentro de un elemento.

    3. 
const contenedor = document.querySelector(".contenedor");
contenedor.innerHTML = "
    Este es un nuevo párrafo
    ";
  3. textContent

    Similar a innerText, pero incluye todo el contenido del nodo, incluso si está oculto mediante CSS.

    4. 
console.log(titulo.textContent);

Modificar atributos

  1. setAttribute

    Añade o modifica un atributo de un elemento.

    2. 
const enlace = document.querySelector("a");
enlace.setAttribute("href", "https://www.ejemplo.com");
  2. getAttribute

    Obtiene el valor de un atributo de un elemento.

    3. 
console.log(enlace.getAttribute("href"));
  3. removeAttribute

    Elimina un atributo de un elemento.

    4. 
enlace.removeAttribute("target");
  4. Atributos directos

    Algunos atributos se pueden modificar directamente a través de propiedades.

    5. 
const imagen = document.querySelector("img");
imagen.src = "nueva-imagen.png";
imagen.alt = "Descripción de la nueva imagen";

Estilos en línea

  1. Modificar estilos directamente

    2. 
const caja = document.querySelector(".caja");
caja.style.backgroundColor = "blue";
caja.style.padding = "20px";
  2. Obtener el valor de un estilo computado 
    3. 
const estilo = window.getComputedStyle(caja);
console.log(estilo.margin); // Muestra el margen aplicado

Clases de los elementos

  1. classList.add

    Añade una o más clases a un elemento.

    2. 
const caja = document.querySelector(".caja");
caja.classList.add("nueva-clase");
  2. classList.remove

    Elimina una o más clases de un elemento.

    3. 
const caja = document.querySelector(".caja");
caja.classList.remove("nueva-clase");
  3. classList.toggle

    Añade la clase si no existe y la elimina si ya existe.

    4. 
const caja = document.querySelector(".caja");
caja.classList.toggle("oculto");
  4. classList.contains

    Comprueba si el elemento tiene una clase específica.

    5. 
const caja = document.querySelector(".caja");
if (caja.classList.contains("nueva-clase")) {
    console.log("La clase está presente");
}

Eventos

Los eventos permiten ejecutar funciones cuando el usuario interactúa con la página (hace clic, pasa el ratón, escribe, etc.).

  1. addEventListener

    Se usa para asociar un evento a un elemento.

    2. 
const boton = document.querySelector("#miBoton");
boton.addEventListener("click", () => {
    console.log("Botón presionado");
});

Eventos comunes

  1. click

    Cuando el usuario hace clic en un elemento.

  2. mouseover

    Cuando el usuario pasa el puntero del ratón por encima de un elemento.

  3. keydown

    Cuando el usuario presiona una tecla.

  4. submit

    Cuando se envía un formulario.

Eliminar eventos


function manejarClick() {
    console.log("Botón presionado");
}
boton.addEventListener("click", manejarClick);
boton.removeEventListener("click", manejarClick);

Crear y añadir elementos al DOM

  1. createElement

    Crea un nuevo elemento HTML.

    2. 
const nuevoParrafo = document.createElement("p");
nuevoParrafo.innerText = "Este es un párrafo nuevo";
  2. appendChild

    Añade un nuevo elemento como hijo de un elemento existente.

    3. 
const contenedor = document.querySelector(".contenedor");
contenedor.appendChild(nuevoParrafo);
  3. removeChild

    Elimina un elemento hijo de un contenedor.

    4. 
contenedor.removeChild(nuevoParrafo);
  4. insertBefore

    Inserta un elemento antes de otro elemento hijo.

    5. 
const primerHijo = contenedor.firstChild;
contenedor.insertBefore(nuevoParrafo, primerHijo);

Fragmentos del DOM

Un DocumentFragment es un contenedor especial que permite agrupar varios nodos antes de insertarlos en el DOM, lo que mejora el rendimiento.


const fragmento = document.createDocumentFragment();

for (let i = 0; i < 5; i++) {
    const item = document.createElement("li");
    item.textContent = `Elemento ${i + 1}`;
    fragmento.appendChild(item);
}

document.querySelector("ul").appendChild(fragmento);
