Tipos de Listas

Volver al indice

Listas ordenadas

Las listas ordenadas muestran los elementos con un número o letra según el orden.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

type: Cambia el tipo de marcador de la lista (1, A, a, I, i)

  1. Elemento A
  2. Elemento B

start: Especifica el número inicial de la lista.

  1. Elemento 5
  2. Elemento 6

Listas no ordenadas

Las listas no ordenadas muestran los elementos con viñetas o marcadores sin un orden específico.

type: Define el tipo de viñeta (disc, circle, square)

  • Elemento cuadrado
  • Otro elemento

Listas de definición

Las listas de definición se utilizan para mostrar términos y sus descripciones.

HTML
Lenguaje de marcado para crear páginas web.
CSS
Lenguaje de estilo para diseñar páginas web.
  • (dt) Define el término.
  • (dd) Define la descripción del término.

Listas anidadas

Las listas pueden anidarse para crear subniveles dentro de la lista principal.

  • Elemento 1
    • Subelemento 1.1
    • Subelemento 1.2
  • Elemento 2
