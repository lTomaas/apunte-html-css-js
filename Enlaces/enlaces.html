Tipos de enlaces

Volver al indice

Enlaces Externos

Conectan a otra página o sitio web fuera del dominio actual.

Para abrir en otra pestaña usamos target _blank

Enlaces Internos

Conectan a una página o recurso dentro del mismo sitio web.

Enlaces a secciones específicas de una página

Permiten saltar a una parte específica dentro de la misma página o en otra página que contenga un identificador.
